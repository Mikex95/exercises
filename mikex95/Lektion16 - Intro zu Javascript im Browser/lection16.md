# Aufgabe

Schau dir die Sammlung an und wähle Experimente aus, das dir am meisten gefällt.
Gehe zum GitHub-Repository deiner drei ausgewählten Experimente und schau dir den Code an, um besser zu verstehen, wie sie erstellt wurden. Stell dir diese Fragen:
Warum gefällt dir dieses Experiment?
Welche Browser-APIs wurden, falls vorhanden, verwendet?
Welche Drittanbieter-Frameworks oder Bibliotheken wurden, falls vorhanden, verwendet?
Welche Drittanbieter-APIs wurden, falls vorhanden, verwendet?
Wird internes oder externes JavaScript in den Dateien verwendet? Wie werden die Skripte geladen?
Wird ein Code-Transpiler verwendet?
Was würdest du ändern, um dieses Experiment noch besser zu machen?
