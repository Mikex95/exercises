# Aufgabe

Beginne, indem du diesen Codepen öffnest: https://codepen.io/DevCraft/pen/MWRxWNO
Du wirst sehen, dass es ein wenig CSS gibt, ein <div> mit einem Label, einem Eingabefeld und einem Button, sowie einer leeren Liste und einem script-Element. Alle deine Ergänzungen machst du im Skript. Also verändere bitte kein HTML, sondern mach alle Änderungen nur über die DOM API in JavaScript.
Erstelle drei Variablen, die auf die Liste (<ul>), das Eingabefeld (<input>) und den Button (<button>) verweisen.
Erstelle eine Funktion, die ausgeführt wird, wenn der Button geklickt wird.
Innerhalb der Funktion speichere zuerst den aktuellen Wert des Eingabefelds in einer Variablen.
Leere anschließend das Eingabefeld, indem du seinen Wert auf einen leeren String ("") setzt.
Erstelle drei neue Elemente - ein Listen-Element (<li>), ein <span> und einen <button> - und speichere sie in Variablen.
Füge das <span> und den Button als Kinder des Listen-Elements hinzu.
Setze den Textinhalt des <span> auf den zuvor gespeicherten Wert des Eingabefelds und den Textinhalt des Buttons auf „Delete”.
Füge das Listen-Element als ein Kind der Liste hinzu.
Nutze die Methode focus(), um das Eingabefeld zu fokussieren, sodass du bereit bist, den nächsten Eintrag für deine Bucket-Liste einzugeben.
Jetzt implementiere noch, dass ein Item auch wieder gelöscht wird, wenn du den Delete-Button drückst.
